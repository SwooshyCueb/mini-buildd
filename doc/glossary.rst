********
Glossary
********

.. glossary::

	Origin
		The "vendor" of a Debian archive.

		Refers to the field of the same name in a Release file.

		Examples: *Debian*, *Ubuntu*.

	Codename
		The name identifying a Debian base distribution.

		Refers to the field of the same name in a Release file.

		Examples: *wheezy*, *jessie*, *sid*, *quantal*.
